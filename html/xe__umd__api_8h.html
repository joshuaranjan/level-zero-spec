◆ xeDriverGetSupportedVersions()

xe_result_t __xecall xeDriverGetSupportedVersions ( uint32_t *  count,
xe_version_t *  versions 
)

Returns the API versions supported by the driver.

If "versions == nullptr", then "count" returns the number of versions supported. Otherwise, "count" specifies the length of "versions" to be reported. In order to only retreive the latest version supported, just pass "count = 1".

Parameters
[in,out]countnumber of versions returned
[out]versionslist of versions supported, from highest to lowest.
